=====================================================
  EmpMonitor Agent — Installation Instructions
=====================================================

FILES IN THIS FOLDER:
  - agent.js       (The monitoring agent)
  - install.bat    (The installer — run this)
  - README.txt     (This file)

HOW TO INSTALL:
  1. Right-click "install.bat"
  2. Click "Run as administrator"
  3. Enter the details when asked:
       - Server URL    : (Get this from your admin)
       - Employee Name : Your full name
       - Admin Password: (Get this from your admin)
  4. Wait for "INSTALLATION COMPLETE!"
  5. Done!

WHAT IT DOES:
  - Runs silently in the background
  - Starts automatically when Windows boots
  - Tracks which apps you use and for how long
  - Takes periodic screenshots
  - Sends data to the company monitoring server

TO UNINSTALL:
  - Run "uninstall.bat" as Administrator
  - Enter the admin password (given by admin)

SUPPORT:
  Contact your system administrator.
